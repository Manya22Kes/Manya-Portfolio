Full Stack Authentication System

Live Demo ↗ GitHub

Overview

A production-grade authentication platform built end-to-end: React + Vite frontend with Context API state management, Node.js/Express backend with 8+ REST API endpoints, and MongoDB user schema with Mongoose.

Auth Flow Architecture

React + Context API
→
Express + 4 Middleware
→
JWT Rotation
+
Google OAuth 2.0

Key Features

  • JWT refresh-token rotation with Axios interceptor queuing
  • Google OAuth 2.0 integration for social login
  • Email verification flow and secure password reset
  • Role-Based Access Control (RBAC) with protected routes
  • 4 reusable Express middleware layers for auth, validation, error handling, and logging
  • Input validation and bcrypt hashing on all sensitive data

Tech Stack

ReactViteNode.jsExpressCloudinaryMongoDBJWTGoogle OAuthbcrypt

Challenges Solved

Token Race ConditionsAxios interceptor queuing ensures concurrent requests wait for token refresh instead of failing.
OAuth + JWT CoexistenceUnified the Google OAuth session and JWT flows into a single, consistent auth state.
